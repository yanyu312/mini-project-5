## Overview
This repository includes tools to generate and visualize audio signals and their spectrograms. It includes:


- `signal_gen.c`: Generates a WAV file with a sequence of tones using different waveform types.
- `spectrogram.c`: Performs short-time Fourier transform on WAV files and outputs ASCII spectrogram data.
- `spectrogram.py`: Visualizes waveform and spectrogram in a two-page PDF.


## File Structure
- `*.wav`: Audio files generated by `signal_gen.c`
- `*.txt`: Spectrogram data output by `spectrogram.c`
- `spectrogram.py`: Python script for visualization
- `Makefile`: For building C executables


## Requirements (for Python only)
```bash
pip install matplotlib numpy
```


## Usage
### Generate Signal (C)
```bash
./signal_gen.exe <sample_rate> <output.wav>
```
Example:
```bash
./signal_gen.exe 8000 s-8kHz.wav
```


### Generate Spectrogram Data (C)
```bash
./spectrogram.exe 32 rectangular 32 10 input.wav output.txt
```
Example:
```bash
./spectrogram.exe 32 rectangular 32 10 s-8kHz.wav s-8kHz.Set1.txt
```


### Visualize with Python
```bash
python3 spectrogram.py <input.wav> <input.txt> <output.pdf>
```
Example:
```bash
python3 spectrogram.py s-8kHz.wav s-8kHz.Set1.txt s-8kHz.pdf
```


## Output
A PDF file with:
1. The waveform of the input audio
2. The spectrogram from the provided text file


## Cleanup
